GuitarFretboard Library Documentation

GuitarFretboard is a customizable guitar fretboard visualization tool that allows developers to create interactive fretboard diagrams, display chord shapes, and play chord progressions.

Getting Started

To use the GuitarFretboard library, include the necessary JavaScript files in your HTML: 

<link rel="stylesheet" href="css/guitar.css">
<script src="js/getNotesInChord.js"></script>
<script src="js/guitar.js"></script>
<script src="js/keyboardCommands.js"></script>

Then, create a container element in your HTML where the fretboard will be displayed: 

<div id="fretboard"></div>

Basic Usage

Initialize a new fretboard with standard tuning and 24 frets: 

const fretboard = new Fretboard("fretboard", ['E', 'A', 'D', 'G', 'B', 'E'], 24);

Display a chord on the fretboard: 

fretboard.highlightChordTones("Cmaj7");

Play a chord progression: 

fretboard.playChords(["G:4", "Bm", "C:8", "D:4"], 100);

Enable keyboard shortcuts: 

listenToKeyPress(fretboard);

Complete Example

<html>

API Reference

Fretboard Constructor

new Fretboard(id, tuning, fretCount)
Parameter Type Description
id String The ID of the HTML element where the fretboard will be displayed
tuning Array An array of strings representing the tuning of each string (e.g., ['E', 'A', 'D', 'G', 'B', 'E'])
fretCount Number The number of frets to display (maximum 32)

Methods

highlightChordTones(chord)

Highlights the notes of a chord on the fretboard.

Parameter Type Description
chord String The chord name (e.g., "C", "Am", "G7", "Dmaj7")

highlightChordInPosition(chord, position)

Highlights a chord in a specific position on the fretboard.

Parameter Type Description
chord String The chord name (e.g., "C", "Am", "G7", "Dmaj7")
position Number The position on the fretboard (1, 2, or 3)

playChords(chords, bpm, enableMetronome)

Plays a sequence of chords at the specified tempo.

Parameter Type Description
chords Array An array of chord strings, optionally with beat counts (e.g., ["G:4", "Bm", "C:8", "D:4"])
bpm Number The tempo in beats per minute
enableMetronome Boolean Whether to enable the metronome sound (default: false)

rotate()

Rotates the fretboard for left-handed view.

flip()

Flips the fretboard vertically.

removeNoteHighlights()

Removes all highlighted notes from the fretboard.

Chord Notation

The library supports various chord types and notations:

Chord Type Notation Example
Major [Root] C, G, D
Minor [Root]m or [Root]min Am, Dmin
Dominant 7th [Root]7 G7, D7
Major 7th [Root]maj7 Cmaj7, Fmaj7
Minor 7th [Root]m7 or [Root]min7 Am7, Emin7
Suspended 4th [Root]sus4 Dsus4, Asus4
Suspended 2nd [Root]sus2 Asus2, Esus2
Diminished [Root]dim Bdim, Fdim
Diminished 7th [Root]dim7 Bdim7, Fdim7
Augmented [Root]aug Caug, Gaug
Add9 [Root]add9 Cadd9, Gadd9
Dominant 9th [Root]9 G9, D9
Minor 9th [Root]m9 or [Root]min9 Am9, Emin9

Keyboard Shortcuts

When keyboard shortcuts are enabled using listenToKeyPress(fretboard), the following shortcuts are available:

Key Action
Space Play/pause chord progression
← (Left Arrow) Go to previous chord
→ (Right Arrow) Go to next chord
Esc Clear the fretboard
r Rotate the fretboard (for left-handed view)
f Flip the fretboard vertically
a-g Set root note for chord building
m Add minor quality to chord
a Add augmented quality to chord
d Add diminished quality to chord
s Add suspended quality to chord
# or b Add accidental to root note
7, 9 Add extension to chord

Note: Chord building with keyboard shortcuts has a 3-second timeout. If no key is pressed within 3 seconds, the chord input is reset.

Examples

Displaying Different Chord Types

// Major chord
fretboard.highlightChordTones("C");

// Minor chord
fretboard.highlightChordTones("Am");

// Dominant 7th chord
fretboard.highlightChordTones("G7");

// Major 7th chord
fretboard.highlightChordTones("Cmaj7");

// Minor 7th chord
fretboard.highlightChordTones("Am7");

// Dominant 9th chord
fretboard.highlightChordTones("G9");

Playing Chord Progressions

// Play a I-IV-V progression in C major (C, F, G) at 120 BPM
fretboard.playChords(["C:4", "F:4", "G:4"], 120);

// Play a ii-V-I progression in C major (Dm7, G7, Cmaj7) at 100 BPM
fretboard.playChords(["Dm7:4", "G7:4", "Cmaj7:8"], 100);

// Play a chord progression with different beat counts
fretboard.playChords(["Am:2", "F:2", "C:2", "G:2"], 80);

Using Positions

// Display C major chord in first position
fretboard.highlightChordInPosition("C", 1);

// Display C major chord in second position
fretboard.highlightChordInPosition("C", 2);

// Display C major chord in third position
fretboard.highlightChordInPosition("C", 3);

Using with Metronome

// Play a chord progression with metronome enabled
fretboard.playChords(["G:4", "Em:4", "C:4", "D:4"], 100, true);
